# 现实参照主动联想规则

## 核心要求
对用户输入中的**每个核心概念**，都要主动尝试给出现实参照方向（至少 2 个方向；信息不足时给检索方向而非强行断言）。

## 参照维度池
- 历史事件/制度
- 真实人物/群体
- 神话/宗教母题
- 民俗/典故
- 文学作品/人物谱系
- 艺术风格（绘画、建筑、雕塑、电影、戏剧）
- 科学机制/假说
- 社会学/心理学解释

## 输出最小格式（每概念）
- 概念：
- 可能参照方向 A/B/C：
- 关联点：
- 可借鉴元素：
- 不应机械照搬：
- 建议保留的原创性：

## 质量约束
- 参照服务于“增厚设定”，不是堆资料。
- 不把参照说成唯一来源。
- 避免高置信度虚构事实；不确定就标“待查证”。
